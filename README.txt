アプリケーションはVisual Studio C# 2013で作成しました。
そのため、.NET Frameworkが必要です。
また、waifu2x本体はVisual C++ 2013 再頒布可能パッケージが必要です。
この二つをMicrosoftよりダウンロードしてきてください。

<!>重要<!>
ディレクトリは相対パスではなく絶対パス指定になっております。後々改良いたしますがファイル群(waifu2xフォルダ丸ごと)は

Cドライブ直下

に入れてください。

実行は waifu2x_gui.exe を実行してください。後は見ての通りです。また、テキストボックスは空欄にしないでください。通常二つとも 2 で大丈夫なはずです。

また、GUI本体はかってに改造してください。
ソース自体、waifu2x_gui_sourceの中に入っています。
後々ライセンスなどは考えます(いまのところWTFPLを予定)
